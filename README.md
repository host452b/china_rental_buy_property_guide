### 口碑方差解读

- **0-1.5 (绿)** = 评价一致，品质稳定
- **1.5-2.5 (黄)** = 多数满意，少数吐槽物业/噪音等
- **2.5-5 (红)** = 争议大，可能因学区变动/物业纠纷/房龄问题

## 数据来源

| 来源 | 用途 |
|------|------|
| 链家/贝壳找房 | 挂牌价、成交价、小区详情 |
| 安居客 | 小区均价、租金、25 项属性 |
| 房天下 | 历史价格走势 |
| 上海市统计局 | 城市价格指数、环线均价 |
| 中国房价行情(禧泰) | 小区级别价格数据 |
| 高德地图 POI API | 小区坐标、地铁站、医院、商业体 |
| 上海市卫健委 | 三甲医院官方名单 |
