# 数据维护流程 (Flow)

## 扩展数据
1. 运行 `scrape_shanghai_communities.py` (需代理IP+高德AMAP_KEY)
2. 合并 `data/` 目录爬取结果到 `shanghai_communities.csv`
3. 去重、补全缺失评分字段

## 校准数据
1. 携程/贝壳/安居客真实评分 → 校准口碑/质量列
2. 高德POI API → 校准地铁/医疗/商业/教育/买菜列
3. 上海统计局价格指数 → 校准价格趋势列
4. Numbeo/GPI → 校准安全列（国际版用）

## 生成产出
1. 重新生成 `shanghai_communities.ipynb` (完整数据红绿色阶)
2. 重新生成 `shanghai_rankings.ipynb` (排行榜)
3. 更新 `README.md` 记录数/覆盖范围

## 列规范 (48列)
- 价格: 2020-2026 × (挂牌均价+成交均价+租金) = 21列
- 评分: 15维 × 0-10分 (含教育资源指数)
- 口碑方差: 0-5 (低=稳定可信)
- 坐标: 经度/纬度 (GCJ-02)
- 5年趋势: 涨/跌/维持
- 新盘交付前年份价格留空
